## 特性
1. 组件扩展自UGUI的Text，代码简单，生成的字体使用内置字体，使用无障碍
2. 使用四通道存储，2048的贴图支持所有GBK的21886个字
3. 支持描边和渐变，描边通过材质设置，渐变和修改字体粗细用脚本
4. 字号20号以上看的比较清楚，小号字需要加描边才能看
